### Setup

1. Clone the repository:
```bash
git clone https://github.com/LeaLiv/HtmlCssProject.git
cd HtmlCssProject
```

2. Open in browser:
   - Simply open `index.html` in your web browser

   ```bash
   npx serve
   ```

3. Access the site:
   - Direct: Open `index.html` with your browser
   - Server: Navigate to `http://localhost:8000`
